# AI Mock Interviewer

An AI-powered interview preparation platform that helps candidates practice technical and HR interviews, receive AI-generated feedback, and generate detailed interview reports.

## Features

* AI-powered mock interviews
* Technical and HR interview modes
* Real-time question generation
* Interview evaluation and scoring
* Detailed feedback reports
* PDF report generation
* User-friendly interface

## Tech Stack

### Frontend

* React.js
* Vite
* CSS

### Backend

* Node.js
* Express.js

### AI Integration

* Google Gemini API

## Project Structure

AI-Mock-INTERVIEWER

├── backend

├── frontend

└── tools

## Installation

### Clone Repository

git clone <repository-url>

### Backend Setup

cd backend

npm install

### Frontend Setup

cd frontend

npm install

### Run Backend

npm start

### Run Frontend

npm run dev
